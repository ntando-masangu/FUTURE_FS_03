# Kicks & Tech Website

A real local business website built for a sneaker & tech reseller online shop in Johannesburg, Gauteng.

## Features
- Home page with hero & tagline
- Product pages with photos, prices (Rand), WhatsApp redirect buttons
- Honest About page (startup story, big goals)
- Working contact form (WPForms + SMTP emails)
- Custom branding (orange buttons, rounded, dark theme)

## Tech
- WordPress + Blosky theme
- Elementor for pages
- WooCommerce for products
- WPForms for contact
- Custom PHP/CSS tweaks for buttons & redirects
